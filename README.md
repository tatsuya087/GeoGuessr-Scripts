<h1 align="center">Scripts List</h1>

<h2 align="center">GeoGuessr Force NMPZ</h2>

A script that forces NMPZ mode for yourself in GeoGuessr. Useful for practicing or challenging yourself in standard modes. You can toggle it ON/OFF from the Tampermonkey menu.

---

<h2 align="center">GeoGuessr Summary Link Glow</h2>

Adds a glow effect to game result links on the GeoGuessr activities page, making them easier to spot.
- **Customizable Color:** You can change the glow color from the Tampermonkey menu ("Change Glow Color üé®").

![How it looks](./GeoGuessr%20Summary%20Link%20Glow/How%20it%20looks.jpg)

---

<h2 align="center">Geoguessr Result Map Resizer</h2>

Allows you to resize the Google Maps embed on the Duels Breakdown page.
- **Adjustable Zoom:** Scale the map from 1.00x to 1.50x using the slider.
- **Draggable Panel:** The control panel can be moved anywhere on the screen.

---

<h2 align="center">Google Street View URL Copy</h2>

Enables you to copy the current URL simply by pressing `Ctrl+C` on Google Street View pages.
- **Smart Copy:** Does not trigger if you have text selected.
- **Notification:** Shows a small toast notification upon successful copy.

---

<h1 align="center">Installation</h1>

1. Install a UserScript manager extension like [Tampermonkey](https://www.tampermonkey.net/).
2. Click on the script file (`.js`) in this repository or visit the GreasyFork link above.
3. Click "Install".
