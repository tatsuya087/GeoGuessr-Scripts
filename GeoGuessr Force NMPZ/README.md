# GeoGuessr 強制NMPZ

GeoGuessrのプレイ中、自分の画面だけを強制的にNMPZモードに制限するスクリプトです。ハンデ戦などに使えます。

## インストール

ブラウザにTampermonkeyを入れた状態で https://greasyfork.org/ja/scripts/552078-geoguessr-force-nmpz を開き、インストールを実行します。

## 機能

ストリートビュー内でのドラッグによる視点移動と、ホイールでのズームをブロックします。コンパスクリックで北を向く操作や、W、A、S、D、Nキーを使った入力も無効化されます。チャットの入力には影響しません。ズームボタンなど一部のコントロール要素は画面から消えます。

Tampermonkeyの拡張機能メニューから、ページを再読み込みせずにスクリプトのオンとオフを切り替えられます。パーティモードのほぼすべてのゲームモードで動きます。ランクやアンランクでは動作しません。

## 使い方

Tampermonkeyアイコンから「GeoGuessr 強制NMPZ」を選び、メニュー内のEnabledかDisabledの項目をクリックして状態を切り替えます。設定した状態はブラウザに保存され、次に開いたときも引き継がれます。

## 注意事項

制限は自分のブラウザ上だけでかかり、他のプレイヤーには影響しません。公式のNMPZモードとは異なり、手元で疑似的に動作を縛る仕組みです。
